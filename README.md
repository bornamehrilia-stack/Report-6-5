# گزارش پروژه: نمایش دما و رطوبت روی LCD با سنسور DHT11 (Temperature & Humidity on LCD with Arduino)

## ۱. مقدمه  
در این پروژه، یک **سنسور DHT11** به آردوینو متصل شده است و هم دمای محیط و هم رطوبت نسبی را اندازه‌گیری می‌کند. سپس این مقادیر به صورت عددی در **نمایشگر LCD 16x2** نمایش داده می‌شوند.

## ۲. اهداف پروژه  
- آشنایی با سنسور DHT11 و نحوه‌ی کار آن  
- یادگیری نحوه‌ی اتصال سنسور DHT11 به آردوینو  
- استفاده از کتابخانه‌ی `DHT.h` برای خواندن داده‌ها  
- نمایش دما و رطوبت روی LCD  
- آشنایی با مفهوم “پروتکل دیجیتال” و “نمایش داده‌های سنسور”

## ۳. قطعات مورد استفاده  
- برد آردوینو Uno  
- سنسور DHT11  
- یک مقاومت ۴.۷k اهم (برای pull-up)  
- نمایشگر LCD 16x2 
- یک پاتنسیومتر ۱۰k اهم (برای تنظیم کنتراست LCD)  
- برد برد (Breadboard)  
- سیم‌های جامپر (jumper wires)



## ۴. نحوه‌ی ساخت مدار  
- **سنسور DHT11**:  
  - VCC → 5V آردوینو  
  - GND → GND آردوینو  
  - DATA → پین دیجیتال ۸ آردوینو  
  - یک مقاومت ۴.۷k اهم بین پین DATA و 5V قرار دهید (برای pull-up).  
- **LCD**:  
  - پین 1 (VSS) → GND  
  - پین 2 (VDD) → 5V  
  - پین 3 (VO) → وسط پاتنسیومتر (برای تنظیم کنتراست)  
  - پین 4 (RS) → پین ۱۲ آردوینو  
  - پین 5 (RW) → GND (در حالت نوشتن)  
  - پین 6 (E) → پین ۱۱ آردوینو  
  - پین 7-10 (D0-D3) → نادیده گرفته می‌شوند (در حالت 4-bit)  
  - پین 11 (D4) → پین ۵ آردوینو  
  - پین 12 (D5) → پین ۴ آردوینو  
  - پین 13 (D6) → پین ۳ آردوینو  
  - پین 14 (D7) → پین ۲ آردوینو  
  - پین 15 (A) → 5V (با مقاومت ۲۲۰ اهم برای روشن کردن پس‌زمینه)  
  - پین 16 (K) → GND  

.

## ۵. کد برنامه (اسکچ):

```cpp
#include <LiquidCrystal.h>   // اضافه کردن کتابخانه LCD
#include <DHT.h>            // اضافه کردن کتابخانه DHT

// تعریف پین‌های LCD
const int rs = 12, en = 11, d4 = 5, d5 = 4, d6 = 3, d7 = 2;
LiquidCrystal lcd(rs, en, d4, d5, d6, d7);  // ایجاد شیء LCD

// تعریف سنسور DHT11
#define DHTPIN 8     // تعریف پین DATA
#define DHTTYPE DHT11 // تعریف نوع سنسور
DHT dht(DHTPIN, DHTTYPE); // ایجاد شیء DHT

void setup() {
  Serial.begin(9600);   // شروع ارتباط سریال برای نمایش داده
  lcd.begin(16, 2);     // شروع LCD با 16 ستون و 2 ردیف
  dht.begin();          // شروع سنسور DHT
  lcd.clear();          // پاک کردن صفحه
}

void loop() {
  delay(1000);  // تأخیر ۱ ثانیه برای ثبات کار

  float h = dht.readHumidity();        // خواندن رطوبت
  float t = dht.readTemperature();     // خواندن دما

  // بررسی خطا در خواندن داده‌ها
  if (isnan(h) || isnan(t)) {
    lcd.setCursor(0, 0);
    lcd.print("failed!");
    return;
  }

  // نمایش رطوبت در ردیف اول
  lcd.setCursor(0, 0);
  lcd.print("H: ");
  lcd.print(h);
  lcd.print(" %");

  // نمایش دما در ردیف دوم
  lcd.setCursor(0, 1);
  lcd.print("T: ");
  lcd.print(t);
  lcd.print(" *C");

  // نمایش داده‌ها در Serial Monitor
  Serial.print("humidity: ");
  Serial.print(h);
  Serial.print(" %");
  Serial.print(" Temperature: ");
  Serial.print(t);
  Serial.println(" *C");
}
```

### توضیح کد:  
- `#include <LiquidCrystal.h>` — اضافه کردن کتابخانه‌ی LCD.  
- `#include <DHT.h>` — اضافه کردن کتابخانه‌ی DHT.  
- `LiquidCrystal lcd(rs, en, d4, d5, d6, d7)` — ایجاد شیء LCD با پین‌های مشخص.  
- `DHT dht(DHTPIN, DHTTYPE)` — ایجاد شیء DHT.  
- `dht.begin()` — شروع سنسور DHT.  
- `dht.readHumidity()` — خواندن رطوبت.  
- `dht.readTemperature()` — خواندن دما.  
- `if (isnan(h) || isnan(t))` — بررسی خطا در خواندن داده‌ها.  
- `lcd.setCursor(0, 0)` — قرار دادن کورسور در ابتدا ردیف اول.  
- `lcd.print()` — نمایش داده‌ها روی LCD.  
- `delay(1000)` — برای اینکه نمایش هر ۱ ثانیه یکبار به‌روز شود.

## ۶. نحوه‌ی کارکرد  
- پس از آپلود کد، سنسور شروع به اندازه‌گیری دما و رطوبت می‌کند.  
- دما و رطوبت به صورت عددی (به واحد سانتی‌گراد و درصد) روی LCD نمایش داده می‌شوند.  
- اگر دست خود را نزدیک سنسور بگیرید، دما و رطوبت افزایش می‌یابند.

## ۷. نتایج  
LCD به درستی دما و رطوبت را اندازه‌گیری و نمایش می‌دهد.
